# usage
This project uses JSON-Server for a mock backend.
# Install Dependencies
npm install
# Run JSON server

The server will run on http://localhost:8000

npm run server
# Run Vite Frontend
React will run on http://localhost:3000

npm run dev
# Build for Production
npm run build
# Preview production build
npm run preview
